# 联想输入法 - 使用说明

## 作业要求实现

✅ **功能1: 数据处理**
- 从新浪新闻数据（JSON格式）提取汉字关联关系
- 逐行读取大型文件（内存友好策略）
- 每个汉字保留前10个最可能的后续字

✅ **功能2: 联想输入法核心**
- 输入一个汉字，返回可能的后续联想字
- 基于频率统计，保留出现次数≥5的后续字
- 数据存储在独立JSON文件中

✅ **功能3: GUI界面**
- 多行文本框：显示输入历史
- 输入框：输入单个汉字
- 标签：显示联想后续字

## 使用说明

### GUI界面操作

1. **输入汉字**: 在"输入汉字"框中输入一个汉字
2. **查看联想**: "联想后续字"标签会显示可能的后续字
3. **确认输入**: 按回车键将输入添加到上方文本框
4. **快速选择**: 输入汉字后，按数字键1-9快速选择对应位置的联想字

### 数据存储策略

**为什么选择每个汉字保留10个后续字？**

1. **存储效率**: 原始数据有7200万字符，处理后仅2MB
2. **覆盖率高**: 前10个后续字覆盖了绝大多数常用组合
3. **响应速度**: 查询速度快，用户体验好
4. **内存友好**: 加载后仅占少量内存

**最小频率阈值（5次）的作用：**
- 过滤掉罕见的错别字或噪声
- 保证联想字的质量
- 减少存储空间
